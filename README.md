## General info
This is the diplom project of training in Yandex Praktikum.

## Technologies
Project is created with:
* Java 11
* JUnit 4.13.2
* maven 3.9.2
* Allure 2.24.0
* RestAssured 4.4.0
* gson 2.8.9
* Selenium 4.8.1

## Setup
To run this project:

```
$ cd Diplom_2
$ mvn compile 
$ mvn clean test
$ mvn allure:serve
```
